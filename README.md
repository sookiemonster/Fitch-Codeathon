# Running the Applications
cd into each folder to run application
1. Server
```
cd server
npm run studio
npm run server
```
  
2. User
```
cd frontend-mobile-user
npm install
npx expo start
```

3. Vendor
```
cd frontend-mobile-vendor
npm install
npx expo start
```

4. Dashboard
```
cd web-client
npm install
npm start
```
